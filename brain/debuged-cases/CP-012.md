# CP-012 — frmExistenciasConReserva breakpoint `Codigo = "01008076"` con `Debug.Print("Espera")`

## 2026-04-29 (@agent) — Apertura

Búsqueda `rg 'Codigo\s*=\s*"[0-9]{8,}"'` en wave 13-8 reveló este sitio en `frmExistenciasConReserva.vb:283`. Para el debugger cuando el código de producto es `"01008076"` durante el cálculo de existencias con reserva.

Particularidad: el breakpoint convive con **dos guards no-op de "asignar a sí mismo"**:

```vb
'If P.Cant_Presentacion = 0 Then
'    P.Cant_Presentacion = 0
'End If

If P.Cant_UMBas = 0 Then
    P.Cant_UMBas = 0
End If
```

El primero está comentado, el segundo está activo. Ambos son **no-ops**: la rama `Then` asigna la propiedad a sí misma. Son **placeholders abandonados** — el desarrollador tenía intención de hacer algo (probablemente recalcular la cantidad cuando es 0) y dejó la estructura armada con cuerpo nulo.
